聚合sdk打包
-----------
这是一个nodejs electron通用的聚合sdk打包工具.<br/>

<h3>实现功能</h3><br />

-  支持自定义配置文件 <br/>

0. 渠道更换 <br/>

1. 游戏名修改 <br/>

2. 自定义包名后缀 <br/>

3. 可扩展图片替换 <br/>

4. 自定义元数据修改 <br/>

5. 环境检测

6. 多进程打包


<h3>实现原理</h3> <br/>
0. 接入接口一致

1.反编绎渠道sdk（这个通常是已反编好的sdk）<br/>

2.反编绎游戏<br/>

3.进行一些预处理（比如：目录建立， 删除）<br/>

4.复制文件<br/>
   - 将sdk中的文件复制到游戏中但要保留一些文件比 <br>
   - values文件夹等文件 <br>

5.合并 xml<br>
   - values中的文件除public.xml外<br>
   - AndroidManifest.xml 
6.回编<br>

7.签名<br>

8.清理<br>
